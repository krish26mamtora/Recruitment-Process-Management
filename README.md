# Recruitment Process Management System (RPMS)

A web-based system to manage the recruitment process, including job postings, candidate handling, interviews, and final selection.

---


## Features

- Job and position management  
- Candidate profile creation and resume screening  
- Interview scheduling (technical and HR rounds)  
- Interview feedback and evaluation  
- Role-based access control (Admin, Recruiter, HR, Interviewer, Candidate)  
- Candidate status tracking and notifications

---

## Tech Stack

| Layer     | Technology              |
|-----------|--------------------------|
| Frontend  | React.js (JavaScript)    |
| Backend   | Spring Boot (Java)       |
| Database  | MySQL                    |
